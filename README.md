# Kalculator

This is a project for [The Odin Project](https://www.theodinproject.com)

Let me present you, a calculator that can
- Add
- Subtract
- Multiply
- Divide
- Exponent

---

This calc has a keyboard support, so u dont have to use mouse to click the button

| Key | Button |
| ----------- | ----------- |
| Enter | = |
| Backspace | Delete |
| Esc | Clear |
